myFlix

myFlix is a movie web application. It allows users to sign up and create a personalised list of their favorite movies, directors and genres

Below is a table explaining the API endpoints

Description URL Method Request Response
Returns a list of all movies /movies GET n/a JSON object of data with all movies. Structure:

{title: 'Your Name', year: 2016, description: "...", genre: "Animation", imageURL: "", director: { name: "Makoto Shinkai", birthYear: 1973, deathYear: "n/a", bio: "....",}

Returns informations about a single movie by title /movies/:title GET n/a JSON object of a single movie same structure as above
Returns information about a genre by name /genre/:name GET n/a JSON object of data about a movie genre; name and description. Example:

{Name: "Drama", Description: "Fiction that is intended to be more serious than comedic"}

Returns information about a director /director/:name GET n/a JSON object of data about a single director. Example:

director: { name: "Makoto Shinkai", birthYear: 1973, deathYear: "n/a", bio: ""...",}

Allows new users to register /users POST JSON object holding data about the user to add. Example:

{ name: "John Kelly" username: "jk123", password: "qwerty123", email: "john@gmail.com", dateofbirth: "1/1/1990" }

JSON object holding data about the user to add. Example:

{ name: "John Kelly" username: "jk123", password: "qwerty123", email: "john@gmail.com", dateofbirth: "1/1/1990" }

Allows users to update their user information /users/:id PUT JSON object holding data about the user to add. Example:

{ name: "John Kelly" username: "jk123", password: "qwerty123", email: "john@gmail.com", dateofbirth: "1/1/1990" }

Text message exlaining that the user's information is updated
Allows users to add a movie to their list of favourites /users/:id/:movie-id POST n/a Text message explaining a movies was added to the user's list of favourites
Remove a movie from a user's list of favourites /users/:id/:movie-id DELETE n/a Text message explaining that a movie was deleted from the user's list of favourites
Remove a user from the registry /users/:id/ DELETE n/a Text message explaining that a the user was removed from the registry
