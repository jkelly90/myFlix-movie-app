myFlix

myFlix is a movie web application. It allows users to sign up and create a personalised list of their favorite movies, directors and genres

Below is a table explaining the API endpoints

Description URL Method Request Response
Returns a list of all movies /movies GET n/a JSON object of data with all movies. Structure:

{
Title: "The Shawshank Redemption",
Description: "Two imprisoned....",
Genre: {
    Name: "Drama"
    Description: "Drama is a genre..."
    },
Director: {
    Name: "Frank Darabont",
    Bio: "Three-time Oscar nominee...",
    Birth: "1959"
    },
ImagePath: "shawshank.png",
Featured: true
}
Returns informations about a single movie by title /movies/:Title GET n/a JSON object of a single movie same structure as above
Returns information about a genre by name /movies/genres/:Name GET n/a JSON object of data about a movie genre; name and description. Example:

{
Name: "Drama",
Description: "Fiction that is intended to be more serious than comedic"
}

Returns information about a director /movies/director/:Name GET n/a JSON object of data about a single director. Example:

{
Name: "Steven Spielberg",
Bio: "One of the most influential...",
Birth: "1946"
}

Allows new users to register /users POST JSON object holding data about the user to add. Example:

{
Name:"vanman",
Password:"movies",
Email:"vanmanfield@gmail.com",
Birthday:"1962-03-22"},
Favorites:[...]
}

JSON object holding data about the user to add. Example:

{
Name:"vanman",
Password:"movies",
Email:"vanmanfield@gmail.com",
Birthday:"1962-03-22"},
Favorites:[...]
}

Allows users to update their user information /users/:Username PUT JSON object holding data about the user to add. Example:

{
Name:"vanman",
Password:"movies",
Email:"vanmanfield@gmail.com",
Birthday:"1962-03-22"},
Favorites:[...]
}

Message exlaining User's information was updated
Allows users to add a movie to their list of favourites /users/:Username/Movies/:MovieID POST n/a Message explaining the selected movie was added to the User's list of favorites
Remove a movie from a user's list of favourites /users/:Username/Movies/:MovieID DELETE n/a Message explaining the selected movie was delete from User's favorites
Remove a user from the registry /users/:Username DELETE n/a Message explaining the selected User was removed from the registry
